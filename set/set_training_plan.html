    查询

    Request

    POST /service/set_training_plan/get

    参数名称 类型 必要性 描述
    SID string R 记录的SID

    Response

    参数名称 类型 必要性 描述
    SID VARCHAR SID
    C_NUMBER VARCHAR 安全培训计划编号
    C_NAME VARCHAR 安全培训计划名称
    B_DELETE BOOLEAN 是否删除
    N_INDEX INTEGER 排序号
    T_CREATE DATETIME 创建时间
    R_CREATE SID 创建人ID
    S_CREATE VARCHAR 创建人
    T_UPDATE DATETIME 修改时间
    R_UPDATE SID 修改人ID
    S_UPDATE VARCHAR 修改人
