    安全报表主页表

    1.新增预赋值

    Request

    POST /service/sr_safety_report/prepare

    不需要条件，预赋值会返回所有的模板名字和sid

    Response

    参数名称 类型 必要性 描述
    DISID VARCHAR — 模板id
    C_TEMPLATE_NAME VARCHAR — 模板名称
    WF_TYPE VARCHAR — 流程状态
    C_ANDROID_URL VARCHAR — 安卓url
    C_IOS_URL VARCHAR — 苹果url
    C_FILE_TYPE VARCHAR — 文件类型

    查询

    Request

    POST /service/sr_safety_report/get

    参数名称 类型 必要性 描述
    SID string R 记录的SID

    Response

    参数名称 类型 必要性 描述
    SID VARCHAR SID
    C_TITLE VARCHAR 标题
    C_YEAR VARCHAR 年度
    C_TABLE_NAME VARCHAR 报表名称
    C_PHONE VARCHAR 联系电话
    T_DATE DATE 填报日期
    R_DEPT SID 填报单位
    R_USER SID 制表人
    R_AUDIT SID 审核人
    R_APPROVER SID 批准人
    R_UNTI_HEAD SID 单位负责人
    R_DEPA_HEAD SID 部门负责人
    R_STATISTICAL_HEAD SID 统计负责人
    C_ENTERPRISE_TYPE VARCHAR 企业类型
    C_QUARTER VARCHAR 季度
    B_DELETE BOOLEAN 是否删除
    N_INDEX INTEGER 排序号
    T_CREATE DATETIME 创建时间
    R_CREATE SID 创建人ID
    S_CREATE VARCHAR 创建人
    T_UPDATE DATETIME 修改时间
    R_UPDATE SID 修改人ID
    S_UPDATE VARCHAR 修改人
