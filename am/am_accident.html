    新增事故事件上报（使用）

    Url

    POST /service/am_accident/addAccidentAndUser

    Request

    参数名称 类型 必要性 描述
    SID VARCHAR SID
    C_NUMBER VARCHAR 事故编号
    C_TIME VARCHAR 事故时间
    C_SITE VARCHAR 事故地点
    R_DICT_PHASES SID 实施阶段id
    R_DICT_TYPE SID 事故类型id
    R_DICT_CATEGORY SID 事故类别id
    R_ORG SID 企业ID
    C_DEATH INTEGER 死亡人数
    C_SLIGHT INTEGER 轻伤人数
    C_SERIOUS INTEGER 重伤人数
    C_DAMAGE VARCHAR 设施设备损坏情况
    C_LOSSES NUMBER 直接经济损失
    C_CAUSE VARCHAR 事故原因
    C_STEPS VARCHAR 事故应急情况及采取的措施
    N_INDEX INTEGER 排序号

    查询

    Request

    POST /service/am_accident/get

    参数名称 类型 必要性 描述
    SID string R 记录的SID

    Response

    参数名称 类型 必要性 描述
    SID VARCHAR SID
    C_NUMBER VARCHAR 事故编号
    C_TIME VARCHAR 事故时间
    C_SITE VARCHAR 事故地点
    R_DICT_PHASES SID 实施阶段id
    R_DICT_TYPE SID 事故类型id
    R_DICT_CATEGORY SID 事故类别id
    R_ORG SID 企业ID
    R_USER SID 用户ID
    C_DEATH INTEGER 死亡人数
    C_SLIGHT INTEGER 轻伤人数
    C_SERIOUS INTEGER 重伤人数
    C_DAMAGE VARCHAR 设施设备损坏情况
    C_LOSSES NUMBER 直接经济损失
    C_CAUSE VARCHAR 事故原因
    C_STEPS VARCHAR 事故应急情况及采取的措施
    N_REPORT CHAR 上报状态
    N_INSTRUCT CHAR 批示状态
    N_SURVEY CHAR 调查报告上传状态
    N_DISPOSE CHAR 处理报告上传状态
    B_DELETE BOOLEAN 是否删除
    N_INDEX INTEGER 排序号
    T_CREATE DATETIME 创建时间
    R_CREATE SID 创建人ID
    S_CREATE VARCHAR 创建人
    T_UPDATE DATETIME 修改时间
    R_UPDATE SID 修改人ID
    S_UPDATE VARCHAR 修改人
