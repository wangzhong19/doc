    查询

    Request

    POST /service/sys_dept/get

    参数名称 类型 必要性 描述
    SID string R 记录的SID

    Response

    参数名称 类型 必要性 描述
    SID VARCHAR SID
    C_CODE VARCHAR 部门编码
    C_NAME VARCHAR 部门名称
    R_PID SID 上级部门
    S_PCODE VARCHAR 上级部门编码
    S_PNAME VARCHAR 上级部门名称
    C_LEVEL VARCHAR 部门级别
    C_TYPE CHAR 机构类型
    N_INDEX INTEGER 排序号
    T_CREATE DATETIME 创建时间
    R_CREATE SID 创建人ID
    S_CREATE VARCHAR 创建人
    T_UPDATE DATETIME 修改时间
    R_UPDATE SID 修改人ID
    S_UPDATE VARCHAR 修改人
