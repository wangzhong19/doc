    查询

    Request

    POST /service/sys_user/get

    参数名称 类型 必要性 描述
    SID string R 记录的SID

    Response

    参数名称 类型 必要性 描述
    SID VARCHAR SID
    C_LOGIN VARCHAR 用户名(全拼)
    C_PASSWORD VARCHAR 密码
    C_SALT VARCHAR 盐
    R_DEPT SID 所属部门
    S_DEPT VARCHAR 所属部门名称
    C_NAME VARCHAR 姓名
    AVATAR VARCHAR 头像
    C_ID VARCHAR 身份证号
    C_PASSPORT VARCHAR 护照号
    C_MAOHAO VARCHAR 冒号
    Z_SEX VARCHAR 性别
    C_MOBILE VARCHAR 移动电话
    C_MAIL VARCHAR 个人邮箱
    C_AUTH VARCHAR 授权情况
    C_ALLOWED VARCHAR 允许作业类别
    C_CERTIFICATE VARCHAR 资质情况
    C_HEALTH_RECORDS VARCHAR 职业健康档案
    B_VALID BOOLEAN 是否可用
    B_DELETE BOOLEAN 是否删除
    N_INDEX INTEGER 排序号
    T_CREATE DATETIME 创建时间
    R_CREATE SID 创建人ID
    S_CREATE VARCHAR 创建人
    T_UPDATE DATETIME 修改时间
    R_UPDATE SID 修改人ID
    S_UPDATE VARCHAR 修改人
