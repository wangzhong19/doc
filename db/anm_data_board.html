中核中原 - API文档

    考核管理数据看板

    预赋值

    Request

    POST /zhzy/dataBoard/anmGetData

    参数名称 类型 必要性 描述
    deptList[] — R 单位数组
    startTime — R 开始时间
    endTime — R 结束时间
    showForm — R 图形代码展现形式 1：列表 0图表
    score — R 筛选条件 点对勾传1

    ###数据格式实例
    “deptList”:[“5cj3y1e1t8k9rrju9kcpx8epmx”,”uxw9w2hgfmkdsu7y484jrc2nw5”],
    “startTime”:”2020-01-19”,
    “endTime”:”2020-01-19”,
    “showForm”:”0”,
    “score”:”1”

    Response

    data[]参数名称 类型 必要性 描述
    id string — 筛选条件id
    dept string — 单位name
    key string — 考核类型name
    val string — 考核分数
    code string — 考核类型code
