    #数据看板-制度化管理

    制度化管理 -hse管理程序

    Request

    POST localhost/zhzy/dataBoard/getHesManagement

    参数名称 类型 必要性 描述
    deptList VARCHAR 选择要查看的部门数组
    showForm VARCHAR 展现形式 1：列表 2：饼状图 3：柱状图 4：曲线图 5：折线图
    startTime VARCHAR 开始时间
    endTime VARCHAR 结束时间

    | totalNumberSystem | VARCHAR | | 制度总数 | 选择了传1，未选不传
    | addNumberSystem | VARCHAR | | 新增数量 |
    | versionNumberSystem | VARCHAR | | 升版数量 |
    | invalidNumberSystem | VARCHAR | | 作废数量 |
    {
    “showForm”:”3”,
    “deptList”:[“utkmmfh18rk64tdrk83289th4y”,”vakfuc8p9phgsrmmttkm5w3aww”,”w4a2auda40h3ruka94pq3xrur4”],
    “totalNumberSystem”:”1”,
    “addNumberSystem”:”1”,
    “versionNumberSystem”:”1”,
    “invalidNumberSystem”:”1”,

    "startTime":"2019-12-01",
"endTime":"2020-04-01"

    }

    Response

    参数名称 类型 必要性 描述
    DEPTNAME VARCHAR 部门名称
    R_DEPT VARCHAR 部门id
    TOTALNUMBERSYSTEM VARCHAR 制度总数
    ADDNUMBERSYSTEM VARCHAR 新增数量
    VERSIONNUMBERSYSTEM VARCHAR 升版数量
    INVALIDNUMBERSYSTEM VARCHAR 作废数量

    制度化管理 法律法规

    Request

    POST localhost/zhzy/dataBoard/getRegulationsMain

    参数名称 类型 必要性 描述
    deptList VARCHAR 选择要查看的部门数组
    showForm VARCHAR 展现形式 1：列表 2：饼状图 3：柱状图 4：曲线图 5：折线图
    startTime VARCHAR 开始时间
    endTime VARCHAR 结束时间
    categorypidList VARCHAR 法律法规类型

    | numberLaws | VARCHAR | | 数量 | 选择了传1，未选不传
    | variousDeptNumberLaws | VARCHAR | | 各部门各类法律法规数量 |

    {
    “showForm”:”1”,
    “deptList”:[“w4a2auda40h3ruka94pq3xrur4”,”utkmmfh18rk64tdrk83289th4y”],
    “numberLaws”:”1”,
    “variousDeptNumberLaws”:”1”,

    "startTime":"2019-12-01",
"endTime":"2020-04-01"

    }

    Response

    参数名称 类型 必要性 描述
    DEPTNAME VARCHAR 部门名称
    R_DEPT VARCHAR 部门id
    NUMBERLAWS VARCHAR 法律法规数量
    VARIOUSDEPTNUMBERLAWS VARCHAR 各部门各类法律法规数量
    R_CATEGORYPID VARCHAR 法律法规类别

    制度化管理 -操作规程

    Request

    POST localhost/zhzy/dataBoard/getOperation

    参数名称 类型 必要性 描述
    deptList VARCHAR 选择要查看的部门数组
    showForm VARCHAR 展现形式 1：列表 2：饼状图 3：柱状图 4：曲线图 5：折线图
    startTime VARCHAR 开始时间
    endTime VARCHAR 结束时间

    | totalNumberOperation | VARCHAR | | 各单位总数量 | 选择了传1，未选不传
    | addNumberOperation | VARCHAR | | 新增数量 |
    | versionNumberOperation | VARCHAR | | 升版数量 |
    | invalidNumberOperation | VARCHAR | | 作废数量 |

    {
    “showForm”:”1”,
    “deptList”:[“w4a2auda40h3ruka94pq3xrur4”,”2sqm4mmc3phhwt9u1xfv053yj1”],
    “totalNumberOperation”:”1”,
    “addNumberOperation”:”1”,
    “versionNumberOperation”:”1”,
    “invalidNumberOperation”:”1”,

    "startTime":"2019-12-01",
"endTime":"2020-04-01"

    ADDNUMBEROPERATION VARCHAR 操作规程新增数量
