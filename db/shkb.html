    生产设备设施列表

    Request

    POST /dataBoard/pf

    参数名称 类型 必要性 描述
    – deptList List 部门数组
    – startTime String 开始时间
    – endTime String 结束时间
    – showForm String 展现形式 1列表3柱形图
    – pfType List 列表不限制，柱形图设备类型最多选5，再给一个全部，如果选择全部字段给我传空
    – pffacility String 设备数量条件，如果勾选传1没勾选传0
    – pfnew String 新增数量条件，如果勾选传1没勾选传0
    – pfout String 退场数量条件，如果勾选传1没勾选传0
    – pfscrap String 报废数量条件，如果勾选传1没勾选传0
    – 柱形图横坐标是部门 一个部门有多个柱子每个柱子是类型，柱子的值是勾选条件的值

    Response

    参数名称 类型 必要性 描述
    – C_TYPE List 设备类型
    – R_SECTION String 部门
    – NEW String 新增设备数量
    – PFOUT List 退场设备数量
    – NOTEXAMINE String 未检查设备数量
    – SCRAP String 报废数量
    – TIME String 时间
    – FPALL String 全部设备数量

    作业安全列表

    Request

    POST /dataBoard/os

    参数名称 类型 必要性 描述
    – deptList List 部门数组
    – startTime String 开始时间
    – endTime String 结束时间
    – showForm String 展现形式 1列表3柱形图
    – osjobclasssid List 列表不限制，柱形图作业类型最多选5，再给一个全部，如果选择全部字段给我传空
    – osconditionarea List 柱形图只能选择一个区域，列表不限制
    – oscount String 柱形图许可数量，如果勾选传1没勾选传0
    – osarea String 柱形图区域，如果勾选传1没勾选传0
    – osstatus String 柱形图状态，如果勾选传1没勾选传0
    – 柱形图横坐标是部门 一个部门有多个柱子每个柱子是类型，柱子的值是勾选条件的值

    Response

    参数名称 类型 必要性 描述
    – R_JOBCLASSSID List 作业类型
    – R_SECTION String 部门
    – TIME String 时间
    – COUNT String 作业数量
    – SETOUT String 作业审批
    – APPROVEING String 作业前检查
    – TURN String 作业后检查
    – APPROVE String 关闭
    – TERMINATION String 终止
    – COUNT(A.R_AREA) String 区域数量
    – 柱形图横坐标是部门 一个部门有多个柱子每个柱子是类型，柱子的值是勾选条件的值

    费用列表

    Request

    POST /dataBoard/cost

    参数名称 类型 必要性 描述
    – deptList List 部门数组 列表不限制，柱形图最多选五个
    – startTime String 开始时间，
    – endTime String 结束时间
    – showForm String 展现形式 1列表3柱形图
    – ttryear String 年份如2020 除了罚款全传年份
    – ttrtype List 费用类型 列表不限制 柱形图值能选择一个或者全部 全部传空
    – ttrstandingbook String 费用台账
    – ttrplan String 预算执行率
    – ttrfine String 罚款台账
    –

    Response

    参数名称 类型 必要性 描述

    – R_TYPEPID String 费用类型
    – C_BUDGETMONEY String 计划费用
    – C_PRACTICALMONEY String 实际费用
    – BUDGET String String 费用执行率
    – FINE String String 罚款
