    查询

    Request

    POST /service/ci_self_category/get

    参数名称 类型 必要性 描述
    SID string R 记录的SID

    Response

    参数名称 类型 必要性 描述
    SID VARCHAR SID
    C_NUMBER VARCHAR 序号
    C_VAL VARCHAR 内容
    C_SCORE INTEGER 分数
    R_SELF SID 上级目录id
    B_DELETE BOOLEAN 是否删除
    N_INDEX INTEGER 排序号
    T_CREATE DATETIME 创建时间
    R_CREATE SID 创建人ID
    S_CREATE VARCHAR 创建人
    T_UPDATE DATETIME 修改时间
    R_UPDATE SID 修改人ID
    S_UPDATE VARCHAR 修改人
