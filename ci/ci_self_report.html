    查询

    Request

    POST /service/ci_self_report/get

    参数名称 类型 必要性 描述
    SID string R 记录的SID

    Response

    参数名称 类型 必要性 描述
    SID VARCHAR SID
    R_ORG SID 单位ID
    R_PROJECT SID 项目id
    C_YEAR VARCHAR 年度时间
    C_SCORE INTEGER 自评总得分
    R_USER_START SID 发起人id
    S_USER_START VARCHAR 发起人name
    R_USER_END SID 审批人id
    S_USER_END VARCHAR 审批人名称
    N_STATUS CHAR 是否审批
    C_TIME VARCHAR 审批时间
    C_OPINION VARCHAR 审批意见
    B_DELETE BOOLEAN 是否删除
    N_INDEX INTEGER 排序号
    T_CREATE DATETIME 创建时间
    R_CREATE SID 创建人ID
    S_CREATE VARCHAR 创建人
    T_UPDATE DATETIME 修改时间
    R_UPDATE SID 修改人ID
    S_UPDATE VARCHAR 修改人
